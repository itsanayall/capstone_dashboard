Introdução

O Capstone foi desenvolvido como projeto final do curso Fundamentals of Data Science in Healthcare, com recurso à linguagem R e encontra-se organizado em várias partes que representam os diferentes módulos leccionados no decorrer do curso.

Contexto e Objetivos

No desenvolvimento deste projeto Capstone, foi utilizada a base de dados “base_gdh_icd10_no_freg.csv”. Este dataset é composto por variáveis relacionadas com perfis de utentes, diagnósticos e características hospitalares, que fornecem uma visão abrangente sobre dados de internamento hospitalar.

Variáveis Utilizadas

De forma a desenvolver um projeto coeso e com o objectivo de tirar partido do máximo de recursos aprendidos ao longo do curso foram selecionadas as seguintes variáveis:

Nome Variável Descrição
Idade idade Esta variável indica a idade do utente e foi selecionada porque possibilita entender o perfil etário dos internamentos e realizar análises segmentadas por faixas etárias;
Sexo sexo Representa o sexo dos utentes e é útil para comparar e identificar possíveis diferenças de diagnóstico ou tipo de admissão.
Tipo de Admissão adm_tip Indica o tipo de admissão por categorias (urgente ou programada) e foi selecionada pela possibilidade de relacionar com outras variáveis de forma a perceber o seu possível impacto.
Grupo de Diagnóstico Homogéneo cod_diagnostico Código que representa o diagnóstico principal de cada utente. A análise desta variável permite observar a prevalência de diagnósticos e investigar as suas associações com outras variáveis.
Código do Hospital hosp_id Código identificador de cada hospital. A inclusão desta variável possibilita a comparação entre diferentes instituições hospitalares.
Distrito distrito Localização geográfica do hospital. Esta variável foi selecionada por possibilitar a realização de análise geográfica por distrito.
Hora de Entrada na Urgência hora_urgencia Indica a hora de entrada no serviço de urgência da instituição de saúde, em segundos e pode ser importante na compreensão de padrões de utilização.

Objectivos da Análise

Os objectivos da análise realizada ao longo deste projecto Capstone são:

  • Caracterizar o perfil dos internamentos hospitalares com recurso ao cruzamento das variáveis selecionadas (idade, sexo, e tipo de admissão);

  • Compreender como o perfil dos internamentos hospitalares influencia o diagnóstico e a localização geográfica.

A concretização destes objectivos pode responder a perguntas como qual a distribuição etária e de sexo dos utentes internados; se existem diferenças significativas no tipo de admissão e nos diagnósticos prevalentes entre diferentes faixas etárias ou entre sexos; e como varia a prevalência de diagnósticos entre diferentes hospitais e distritos.

A análise destas questões permite uma visão mais pormenorizada sobre padrões de internamento hospitalar e as suas variações de acordo com as características dos utentes e dos hospitais, que podem ser insights importantes para melhorar a eficiência da gestão hospitalar e da alocação de recursos.

# Dividir os dados em treino e teste
set.seed(123)
train_index <- createDataPartition(data_without_na$mortalidade_apr31, p = 0.7, list = FALSE)
train_data <- data_without_na[train_index, ]
test_data <- data_without_na[-train_index, ]
# Treinar o modelo de Random Forest
library(randomForest)
modelo_ml <- randomForest(mortalidade_apr31 ~ idade + adm_tip , 
                                      data = train_data,
                                      ntree = 100)
# Fazer predições no conjunto de teste
predicoes <- predict(modelo_ml, newdata = test_data)

# Avaliar o modelo
conf_matrix <- confusionMatrix(predicoes, test_data$mortalidade_apr31)
